# <a name="unable-to-print"></a>Nelze tisknout.

Jsme si vědomi problému, kdy někteří uživatelé nemohou tisknout při používání prohlížečů Safari nebo Edge. Zatímco pracujeme na opravě, použijte jiný prohlížeč. Omlouváme se za nepříjemnosti.
