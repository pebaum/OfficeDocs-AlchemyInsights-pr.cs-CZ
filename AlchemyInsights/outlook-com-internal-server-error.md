# <a name="internal-server-errors-in-outlookcom"></a>Interní server chyby v Outlook.com

O žádné! Pokud se zobrazí zpráva "Vnitřní chyba serveru" nebo "Chyba 500", je nám a ne - bohužel o který.

Na [stránce Stav služby Office](https://portal.office.com/servicestatus) zkontrolujte nic jinak děje, ale nejlepším řešením je udělat pauzu několik minut a potom stisknutím kombinace kláves **CTRL** a **F5**aktualizujte prohlížeč.
