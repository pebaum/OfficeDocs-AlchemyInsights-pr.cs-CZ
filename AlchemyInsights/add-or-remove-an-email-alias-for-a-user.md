# <a name="add-or-remove-an-email-address-for-a-user"></a>Přidat nebo odebrat e-mailovou adresu uživatele

[Chcete-li začít, přejděte sem](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions).

 **Poznámka**: máte více než jednu e-mailovou adresu uživatele, ale další *e-mailové aliasy* lze použít pouze pro příjem e-mailu. Pokud potřebujete odeslat e-mail s aliasem, vytvořte [sdílené poštovní schránky](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9). Tímto způsobem dosáhnete alias pro odesílání a přijímání pošty.
