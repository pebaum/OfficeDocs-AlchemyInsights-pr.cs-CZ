# <a name="how-to-access-outlookcom-while-traveling"></a>Jak získat přístup k Outlook.com při cestování

Chcete-li získat přístup k Outlook.com při cestování, zkontrolujte, zda je aktuální informace o zabezpečení účtu Microsoft - alternativní e-mailovou adresu nebo telefonní číslo kde získat kódy zabezpečení. Další informace naleznete v tématu [přístup k Outlook.com při cestování](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy).
