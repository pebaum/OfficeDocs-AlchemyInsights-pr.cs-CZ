# <a name="set-up-spf-in-office-365"></a>Nastavení SPF ve službách Office 365

Aktualizujte záznamy DNS tak, aby zásad Framework odesílatele (PSPM) s vaší vlastní domény můžete použít k ověření odchozího e-mailu odeslány z vaší vlastní domény. To zabraňuje ostatním uživatelům "falešného" mail jako by byly odeslány z vaší domény.
  
- [Nastavení specifikace SPF v Office 365 jako prevence falšování identity](https://docs.microsoft.com/en-us/office365/SecurityCompliance/set-up-spf-in-office-365-to-help-prevent-spoofing).
    
- [Nápovědu nastavení všechny záznamy DNS](https://docs.microsoft.com/en-us/office365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)
