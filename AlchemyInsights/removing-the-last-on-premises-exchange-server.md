# <a name="removing-the-last-on-premises-exchange-server"></a>Odebrání posledního místního serveru Exchange

Před odebráním posledního serveru Exchange z místní organizace byste se měli ujistit, že rozumíte důsledkům tohoto problému, abyste se rozhodli, zda tento poslední server potřebujete pro jiné účely. Další informace najdete v tom, [jak a kdy vyřadit z provozu místní servery Exchange v hybridním nasazení](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
