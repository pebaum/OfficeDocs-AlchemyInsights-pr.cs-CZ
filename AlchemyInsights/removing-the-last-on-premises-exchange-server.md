# <a name="removing-the-last-on-premises-exchange-server"></a>Odebrání posledního místního serveru Exchange

Před odebráním serveru Exchange poslední z místní organizace, nezapomeňte, že chápete důsledky tím vám pomohou rozhodnout, pokud potřebujete poslední server pro jiné účely. Další informace rezervovat [jak a kdy chcete vyřadit z provozu serverů Exchange v prostorách v hybridní nasazení](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
