# <a name="solutions-for-unsupported-operating-system"></a>Řešení pro "Nepodporovaný operační systém"

Při pokusu o instalaci microsoftu 365 se může zobrazit chybová zpráva **nepodporovaného operačního systému,** pokud vaše zařízení nesplňuje [systémové požadavky](https://products.office.com/office-system-requirements).
  
Microsoft 365 vyžaduje:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 nebo Windows Server 2008 R2

    NEBO

- Mac OS X 10.10

Pokud se tato zpráva dostala při pokusu o instalaci Office na mobilní zařízení nebo Chromebook, přečtěte si téma [Jak nainstalovat a spustit Microsoft Office v Chromebooku](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
