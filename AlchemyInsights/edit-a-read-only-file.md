# <a name="edit-a-read-only-file"></a>Upravte soubor jen pro čtení

V knihovnách dokumentů po klepnutí na název dokumentu sady Office může neočekávaně otevřít jej jako soubor jen pro čtení. Otevřete pro úpravy, klepněte na tlačítko **Otevřít** (tři tečky) a pak klepněte na tlačítko **Upravit**
