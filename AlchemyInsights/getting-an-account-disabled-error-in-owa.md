# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Jak se chyba účtu v Outlooku na webu zobrazuje v chybě?

Pokud se při pokusu o `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` otevření Outlooku na webu (dříve označované jako Outlook Web App) zobrazí chyba Váš účet byl **zakázán,** správce organizace vám mohl zakázat přístup k Outlooku na webu. Správce může tuto chybu opravit podle kroků v tématu [Povolit nebo zakázat aplikaci Outlook Web App pro poštovní schránku](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
