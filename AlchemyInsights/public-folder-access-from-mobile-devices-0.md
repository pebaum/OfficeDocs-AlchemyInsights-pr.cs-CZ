# <a name="public-folder-access-from-mobile-devices"></a>Přístup k veřejným složkám z mobilních zařízení

Veřejné složky nejsou přístupné z mobilních zařízení.
  
Pokud chcete mít funkce podobné k veřejným složkám a přístup z mobilních zařízení, naleznete [Další informace o skupinách Office 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) lepší alternativou.
