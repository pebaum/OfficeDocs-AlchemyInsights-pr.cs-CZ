# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Použití SaRA k diagnostice a řešení problémů s ověřováním v aplikaci Outlook

Důrazně doporučujeme používat [aplikaci Outlook a požádat](https://aka.ms/SaRA-OutlookPwdPrompt-Alchemy) o diagnostiku hesla v ohroženém počítači, aby vyřešili problémy, při kterých aplikace Outlook neustále zobrazuje výzvu k zadání hesla. Diagnostika [SaRA](https://diagnostics.office.com/#/) provádí automatizované kontroly a vrací možná řešení, která lze použít k řešení zjištěných problémů.
