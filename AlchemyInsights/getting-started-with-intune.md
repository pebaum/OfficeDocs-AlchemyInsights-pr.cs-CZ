# <a name="getting-started-with-intune"></a>Začínáme s Intune

Pro rychlou prohlídku Intune a konzole pro správu najdete [v návodu k Intune na Webu Azure Portal](https://docs.microsoft.com/intune/fundamentals/tutorial-walkthrough-intune-portal).

Pokud chcete vyzkoušet Intune nebo nastavit zkušební verzi, přečtěte si [úvodní příručku: Vyzkoušejte Microsoft Intune zdarma](https://docs.microsoft.com/intune/fundamentals/free-trial-sign-up).

Pokud už Intune máte a jste připraveni ho začít používat, přečtěte si průvodce [Nastavením Intune.](https://docs.microsoft.com/intune/fundamentals/setup-steps)
