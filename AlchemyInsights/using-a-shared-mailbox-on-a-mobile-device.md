# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Použití sdílené poštovní schránky na mobilním zařízení

Chcete-li v mobilním zařízení používat sdílenou poštovní schránku, měli byste [si stáhnout Outlook mobile](https://products.office.com/outlook-mobile-for-android-and-ios) a potom přidat [sdílenou poštovní schránku do Outlooku mobile](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f).
  
Jak získat nápovědu: [Získejte nápovědu k aplikaci pro Outlook pro iOS a Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
