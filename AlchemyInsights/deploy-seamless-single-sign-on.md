# <a name="enable-seamless-sso"></a>Povolit bezproblémový SSO

Azure Active Directory (Azure AD) bezproblémové jednotné přihlášení (bezproblémový zápis SSO) automaticky podepíše uživatele, pokud se nacházejí v podnikových počítačích připojených k podnikové síti.
  
[Chcete-li povolit bezproblémový přihlášení SSO, postupujte podle kroků v tomto článku.](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
