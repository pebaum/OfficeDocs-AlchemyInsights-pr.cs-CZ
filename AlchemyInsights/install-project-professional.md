# <a name="install-project-professional"></a>Instalovat aplikaci Project Professional

Viz [Instalace aplikace Project](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281?ui=en-US&rs=en-US&ad=US) se dozvíte, jak nainstalovat verzi předplatného aplikace Project Online a bez odběru verzi projektu 2019, 2016 nebo 2013. 

Aplikace Project 2010 nebo 2007, postupujte podle kroků v [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) nebo [systému Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Projekt je prodáván jako samostatnou aplikaci, takže pokud jste ji dosud, [koupit nebo zkuste projektu](https://products.office.com/project).
