# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Chyba v aplikaci Outlook na webu získávání účet zakázán?

Pokud se zobrazí chybová zpráva, **váš účet byl zakázán** s `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` při pokusu o otevření aplikace Outlook na webu (dříve známá jako aplikace Outlook Web App) organizace správce zakázal přístup k aplikaci Outlook na webu. Správce můžete tuto chybu opravit podle kroků uvedených v tématu [Povolení nebo zakázání aplikace Outlook Web App poštovní schránky](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
