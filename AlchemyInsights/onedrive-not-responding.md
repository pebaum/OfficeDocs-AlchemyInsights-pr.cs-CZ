# <a name="onedrive-not-responding"></a>Funkce OneDrive nereaguje.

U nově vytvořených účtů se uživatelé musí přihlásit ke středisku Microsoft 365 Admin Center s vlastním účtem a klepnutím na tlačítko OneDrive ručně spustit poskytování webu OneDrive.
  
Pokud však uživatel již klepnul na položku OneDrive a jejich osobní web není zajištěn do 24 hodin, zadejte do protokolu lístek odborné pomoci.
