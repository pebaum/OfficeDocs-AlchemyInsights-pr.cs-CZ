# <a name="how-to-type-special-characters-on-a-mac"></a>Jak psát speciální znaky na Mac

Chcete-li zadat speciální znaky v počítači Mac, otevřete nabídku **speciálních znaků** klávesovou zkratkou ⌘ **příkaz** + **ovládací prvek** + **prostoru**.

Rychlé zkratky pro @ je také **ALT** + **64**.
