# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>550 5.7.708 Přístup odepřen, provoz nebyl přijat z této IP adresy

K této chybě dochází při odesílání e-mailů ze známých ip adres s nízkou reputací, které obvykle používají noví zákazníci.

Pokud musíte před zakoupením licencí odeslat e-mail z těchto IP adres, požádejte o výjimku, dokud nebudete moct zakoupit licence.
