# <a name="create-and-use-a-shared-mailbox"></a>Vytvořit a používat sdílené poštovní schránky

Sdílené poštovní schránky umožňuje více osobám odesílat a přijímat e-maily pro alias, stejně jako **sales@contoso.com**, bez přiřazení jinou licenci.
  
[Vytvořit sdílené poštovní schránky](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) vysvětluje, jak: 
  
- Vytvoření sdílené poštovní schránky
    
- Uživatelům udělit oprávnění k použití sdílené poštovní schránky
    
- Přístup ke sdílené poštovní schránky v aplikaci Outlook nebo Outlook na webu
