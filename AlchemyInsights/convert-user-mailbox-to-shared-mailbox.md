# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Převést poštovní schránky uživatele poštovní schránku místnosti nebo vybavení

Poštovní schránky můžete také převést z jedné poštovní schránky typu na jiný, ale budete muset provádět tyto úkoly pomocí Exchange Online prostředí PowerShell. Rezervujte [Převést poštovní schránky](https://go.microsoft.com/fwlink/p/?LinkId=832875) se další podrobnosti.
