# <a name="cant-download-files-in-chrome"></a>Nelze stáhnout soubory v Chrome

**Problém** Jsme si vědomi tohoto problému, který brání někteří zákazníci moci stáhnout soubory, při použití prohlížeče Google Chrome. 

**Řešení** Používejte [Microsoft okraj](https://www.microsoft.com/en-gb/windows/microsoft-edge) nebo jiný webový prohlížeč.
Další informace o [stahování souborů a příloh na Outlook.com](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab).
