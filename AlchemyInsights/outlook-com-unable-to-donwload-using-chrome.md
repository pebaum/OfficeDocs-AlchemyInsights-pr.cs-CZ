# <a name="cant-download-files-in-chrome"></a>V Chromu nelze stahovat soubory

**Problém** Jsme si vědomi problému, který některým zákazníkům brání v stahování souborů při používání prohlížeče Google Chrome. 

**Řešení** Použijte [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) nebo jiný webový prohlížeč.
Přečtěte si další informace o [stahování souborů a Outlook.com příloh .](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy)
