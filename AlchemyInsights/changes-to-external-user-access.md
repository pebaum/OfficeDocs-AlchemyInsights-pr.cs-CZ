<span data-ttu-id="82e92-p101">Od 23. března 2018 externí uživatelé nebudou nadále přístup k obsahu, která byla sdílena s všechny All Authenticated Users a všichni uživatelé formuláře, i když byl pozván nebo přidány do skupiny oprávnění. Externí uživatelé pouze budou moci získat přístup k obsahu, která byla sdílena s uživatele nebo skupiny, do kterých uživatel patří. Pokud chcete zachovat stávající nastavení, naleznete v tématu:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365</span><span class="sxs-lookup"><span data-stu-id="82e92-p101">Starting March 23, 2018, external users will no longer be able to access content that has been Shared with Everyone, All Authenticated Users, or All Forms Users, even if they have been invited or added to authorized groups. External users will only be able to access content that has been shared with the user or with groups to which the user belongs. If you would like to keep your existing settings, please see: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365</span></span>
