# <a name="how-to-select-all-in-outlookcom"></a>Jak k "Vybrat vše" v Outlook.com

Můžete pouze vybrat všechny Pokud jsou zobrazeny všechny zprávy. Toto je zabránit náhodnému akce na zprávy, které nelze zobrazit.

**CTRL + klepnutí** vybere více jednotlivých zpráv.

**Stisknutím klávesy SHIFT a klepnutím** vyberete všechny zprávy mezi první a poslední zprávu, kterou jste vybrali.
