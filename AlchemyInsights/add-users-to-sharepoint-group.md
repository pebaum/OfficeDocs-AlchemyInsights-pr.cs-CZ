# <a name="add-users-to-a-sharepoint-group"></a>Přidání uživatelů do sharepointové skupiny

Skupina SharePoint je kolekce uživatelů, kteří mají stejnou sadu oprávnění k webům a obsahu. Místo přiřazování oprávnění po jedné osobě můžete pomocí skupin pohodlně přiřadit stejnou úroveň oprávnění mnoha lidem najednou.

- [Přidání uživatelů do skupiny](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#add-users-to-a-group)
- [Přidání nebo odebrání členů ze skupin Microsoftu 365 pomocí Centra pro správu](https://docs.microsoft.com/microsoft-365/admin/create-groups/add-or-remove-members-from-groups)
