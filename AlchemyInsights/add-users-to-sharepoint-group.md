# <a name="add-users-to-a-sharepoint-group"></a>Přidání uživatelů do skupiny SharePointu

Skupina SharePoint je kolekce uživatelů, kteří mají stejnou sadu oprávnění k webům a obsahu. Místo toho, abyste přiřazována oprávnění jednu osobu najednou, můžete použít skupiny pohodlně přiřadit stejnou úroveň oprávnění pro mnoho lidí najednou.

- [Přidání uživatelů do skupiny](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#add-users-to-a-group)
- [Přidání nebo odebrání členů ze skupin Microsoftu 365 pomocí Centra pro správu](https://docs.microsoft.com/office365/admin/create-groups/add-or-remove-members-from-groups?view=o365-worldwide)
