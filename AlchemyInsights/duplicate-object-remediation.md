# <a name="duplicate-object-synchronization-errors"></a>Duplicitní objekt chyby synchronizace

Pokud se zobrazí jedna z následujících chyb během synchronizace:

- Synchronizovaného objektu s stejnou adresu proxy serveru již existuje v adresáři služby Microsoft Online Services.

- Podobná zpráva UserPrincipalName.

Chcete-li vyřešit duplicitní objekt chyby pomocí nových diagnostických AADConnect zdraví. Další informace naleznete v tématu [Diagnóza a napravovat chyby synchronizace duplicitní atribut](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
